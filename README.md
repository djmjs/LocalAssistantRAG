# chatbot_local_withLangChain

Minimal local chatbot project using LangChain-style tooling.

Purpose
- Provide a small workspace that demonstrates embedding and local vector storage and a simple main entrypoint.

Quick start
1. Create a Python virtual environment and activate it:

   powershell

   python -m venv .venv
   .\.venv\Scripts\Activate.ps1

2. Install dependencies:

   pip install -r requirements.txt

3. Run the app:

   python main.py

Sharing to GitHub
- Create a new repository on GitHub (via web UI or `gh repo create`).
- Add the remote and push (example commands are provided below in the repository root instructions).
